# flappy-bird
Flappy bird game implemented in Python with Pygame library

# dependencies
<ul>
  <li>python3</li>
  <li>pygame</li>
</ul>

# installation

    - ```git clone https://github.com/jbcallv/flappy-bird.git```
<ol>
  <li>install all dependencies</li>


  <li>cd flappy-bird</li>
  <li>python3 main.py</li>
</ol>
